# Sync 模块

## 1. 模块定位

* **职责**：提供跨平台的同步原语接口，用于多线程编程中的同步和互斥。
* **边界**：不处理高级并发模式或复杂同步机制，仅提供基础同步原语。
* **外部依赖**：无

## 2. 设计

采用模板化设计，提供类型安全的同步原语，支持多种内存顺序和同步策略。

## 3. API

* **Atomic**：原子操作封装，支持加载、存储、交换、比较交换、增减等操作。
* **Backoff**：退避策略，用于减少自旋锁的CPU占用。
* **CacheLine**：缓存行对齐相关功能，减少伪共享。
* **Cancel**：取消机制，用于取消正在执行的任务。
* **Fence**：内存屏障，用于控制内存操作的顺序。
* **MemoryOrder**：内存顺序枚举，定义原子操作的可见性和顺序约束。
* **ScopedLock**：作用域锁，自动管理锁的获取和释放。
* **SpinLock**：自旋锁实现，用于短时间的互斥访问。
* **TaskScope**：任务作用域，用于管理任务的生命周期和同步。
