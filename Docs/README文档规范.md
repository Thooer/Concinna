## 3. 模块结构

<ModuleName>/
├── API/                        # (必选)接口
├── Impl/                       # (可选)实现目录
│   └── <ImplName>/             #       (可选)接口的实现
├── Sample/                     # (可选)示例与测试
│   └── <SampleName>/           #       (必选)示例/测试
├── SubModule/                  # (可选)嵌套子模块
│   └── <SubmoduleName>/        #       (嵌套)遵循标准结构
├── Docs/                       # (可选)文档
├── README.md                   # 模块设计与使用说明
└── CMakeLists.txt              # 模块构建配置

* **依赖**：列出直接依赖的模块（不展开其内容）。
